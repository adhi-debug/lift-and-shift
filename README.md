# DOM Event Handling Demo

## 📌 Overview

This project demonstrates different ways of registering events in JavaScript using the Document Object Model (DOM). It shows how user interactions like button clicks can trigger functions.

---

## 🚀 Features

* Inline event handling
* DOM property-based event handling
* addEventListener method (modern approach)
* Simple and beginner-friendly implementation

---

## 🛠️ Technologies Used

* HTML
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

* index.html → Contains HTML structure and JavaScript code

---

## ▶️ How to Run

1. Download or copy the code
2. Open the file in any web browser
3. Click the buttons to see event handling in action
4. Check browser console (F12) for output

---

## 📖 Concepts Covered

* DOM (Document Object Model)
* Event handling
* Event listeners
* User interaction handling

---

## ⚡ Methods Demonstrated

### 1. Inline Event Handling

Event is written directly inside HTML.

### 2. DOM Property Method

Event is assigned using JavaScript.

### 3. addEventListener

Modern way to attach multiple events.

---

## ✅ Conclusion

This project helps understand how JavaScript interacts with HTML elements using different event registration methods. It is useful for beginners learning frontend development.
